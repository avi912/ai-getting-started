# ai-getting-started

![Claude Node CI](https://github.com/avi912/ai-getting-started/actions/workflows/claude-node.yml/badge.svg)

This repository contains shell-learning materials and examples. See examples/claude-node for a Node.js example that demonstrates calling Anthropic Claude via HTTP.

- examples/claude-node: minimal Node 18 example, requires ANTHROPIC_API_KEY (and optional ANTHROPIC_ADMIN_API_KEY) stored as repository secrets for CI.

See .github/copilot-instructions.md and CLAUDE.md for guidance on using AI assistants with this repo.
